Porsche factory paint codes for the 944, 944 Turbo and 944 S, by model year. The code (e.g. `LM 3 A`) is the factory paint number — useful when ordering touch-up or matching paint.

**Standard Colors**

| Color | Code |
| --- | --- |
| Mocca Black | `LM 9 V` |
| Gabun Gray | `LY 7 A` |
| Guards Red | `LM 3 A` |
| Gambia Red | `LA 3 B` |
| Mauritius Blue | `LY 5 A` |
| Alpine White | `L 90 E` |
| Havana Brown | `LD 8 A` |
